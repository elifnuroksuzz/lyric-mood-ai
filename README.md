# Lyric Mood AI

Yapay zeka teknolojileriyle şarkı sözlerindeki duygusal içerikleri analiz eden Streamlit tabanlı bir Python uygulaması. Genius API üzerinden şarkı sözlerini otomatik olarak çeker ve Groq AI ile mutluluk, hüzün, öfke, korku ve aşk gibi beş duygu kategorisinde detaylı bir analiz yapar.

## 🎵 Özellikler

- **Otomatik Şarkı Sözü Çekme**: Genius API kullanarak şarkı sözlerini otomatik olarak getirir
- **Duygu Analizi**: Groq AI ile gelişmiş duygu analizi
- **5 Duygu Kategorisi**: Mutluluk, Hüzün, Öfke, Korku ve Aşk kategorilerinde analiz
- **Streamlit Arayüzü**: Kullanıcı dostu web tabanlı arayüz
- **Detaylı Raporlama**: Her duygu kategorisi için yüzdelik analiz

## 🛠️ Teknolojiler

- **Python**: Ana programlama dili
- **Streamlit**: Web arayüzü
- **Genius API**: Şarkı sözü çekme
- **Groq AI**: Duygu analizi
- **Natural Language Processing**: Metin işleme

## 🚀 Kurulum

1. Repository'yi klonlayın:
```bash
git clone https://github.com/elifnuroksuzz/lyric-mood-ai.git
cd lyric-mood-ai
```

2. Gerekli paketleri yükleyin:
```bash
pip install -r requirements.txt
```

3. API anahtarlarınızı yapılandırın:
   - Genius API anahtarı
   - Groq AI API anahtarı

4. Uygulamayı çalıştırın:
```bash
streamlit run app.py
```

## 📱 Kullanım

1. Uygulama arayüzünde şarkı adını ve sanatçıyı girin
2. "Analiz Et" butonuna tıklayın
3. Şarkı sözleri otomatik olarak çekilir
4. AI destekli duygu analizi sonuçlarını görüntüleyin
5. Her duygu kategorisi için detaylı yüzdelik değerleri inceleyin

## 📊 Duygu Kategorileri

- **Mutluluk**: Pozitif duygular ve neşe
- **Hüzün**: Melankolik ve üzüntülü duygular
- **Öfke**: Agresif ve kızgın duygular
- **Korku**: Endişe ve kaygı duyguları
- **Aşk**: Romantik ve sevgi duyguları

## 🔧 API Yapılandırması

Uygulama çalışması için aşağıaki API anahtarlarına ihtiyaç duyar:

- **Genius API**: Şarkı sözlerini çekmek için
- **Groq AI API**: Duygu analizi için
